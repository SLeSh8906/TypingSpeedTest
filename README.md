# Typing Speed Test (C# Edition)

## Опис
Консольний застосунок на .NET 8 для перевірки швидкості друку.
Користувач має ввести запропонований текст на швидкість.

## Мета
Навчитися працювати з Git flow, гілками, SSH ключами та створити
базовий додаток на C# для роботи з рядками та часом.

## Контрольні питання
1. **git init** - створює новий репозиторій.
2. **dotnet run** - компілює та запускає C# програму.
3. **SSH** - використовується для безпечного з'єднання з GitHub без пароля.
